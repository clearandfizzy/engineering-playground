# Engineering Playground

A lightweight repository for a senior full-stack engineer to practice small, daily software engineering exercises.

## Purpose

This playground is designed for long-term growth with practical, self-contained challenges across backend, frontend, testing, architecture, devops, and AI engineering.

## Repository structure

- `api/`
- `frontend/`
- `testing/`
- `architecture/`
- `devops/`
- `ai/`
- `docs/`
- `challenges/`

## Challenge principles

Each challenge should:

- take 30–60 minutes
- be completely self-contained
- demonstrate a real software engineering skill
- remain runnable independently
- favour practical engineering over algorithm puzzles

## Tooling

- Node.js (latest LTS)
- TypeScript
- ESLint
- Prettier
- EditorConfig

## npm scripts

- `npm run lint`
- `npm run format`
- `npm run build`

This setup is intentionally minimal so it is easy to extend with future React, Next.js, Express, Docker, Playwright, and AI examples.
